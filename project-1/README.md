# Travel Booking Website

## Project Overview
A fully responsive, accessible travel booking website built with semantic HTML, CSS, and JavaScript.

## Work Done in index.html

### 1. **Header Section**
- Semantic `<header>` with `role="banner"`
- Navigation bar with `aria-label="main navigation"`
- Proper heading hierarchy

### 2. *Main Section**
- Full-width background video with overlay
- Compelling call-to-action (CTA) button
- Accessible link with `aria-label`

### 3. **Destinations Section**
- Grid layout displaying popular travel destinations
- Images with proper alt text
- Semantic `<section>` with ARIA labels

### 4. **Packages Section**
- Package cards with pricing and descriptions
- "Book Now" buttons for each package
- Responsive grid layout

### 5. **Testimonials Section**
- Customer reviews and ratings
- Accessible list structure with `role="list"`
- Responsive grid display

### 6. **Booking Form Section**
- Complete booking form with fields:
  - Full Name (required)
  - Email (required)
  - Phone Number (required)
  - Destination (dropdown)
  - Message (optional textarea)
- Form submission handler with success alert
- Form validation and reset functionality

### 7. **Footer Section**
- Footer links navigation
- Social media links
- Contact information with Font Awesome icons:
  - `fa-map-marker-alt` - Location
  - `fa-phone` - Phone
  - `fa-envelope` - Email
- Footer bottom with copyright & legal links

### 8. **Accessibility Features**
- Semantic HTML5 elements (`<header>`, `<nav>`, `<section>`, `<footer>`)
- ARIA labels and roles for screen readers
- Proper heading hierarchy (h1, h2)
- Alt text for images
- Form labels with proper associations

### 9. **JavaScript Functionality**
- Form submission handler
- Success alert on form submit
- Form validation
- Auto-reset after submission

## Key Features
✅ Fully Responsive (Mobile, Tablet, Desktop)
✅ Accessible (WCAG compliant)
✅ Font Awesome Icons integration
✅ Google Fonts (Poppins)
✅ Clean, semantic HTML structure
✅ Interactive form with validation

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome Icons
- Google Fonts

## Responsive Breakpoints
- Desktop: 1024px+
- Tablet: 768px - 1023px
- Mobile: 480px - 767px
- Extra Small: Below 360px
